# DISH Network

This repository tracks APIs, SDKs, and other developer resources for DISH Network, now part of EchoStar following the 2024 merger that brought DISH and Hughes back together under one parent.

DISH Network is a U.S. television provider offering satellite and streaming TV services, plus wireless services through its Boost Mobile and DISH Wireless brands. The Boost Mobile and DISH Wireless networks are operated on a 5G Open RAN architecture.

## APIs

No public consumer-facing developer APIs have been documented.
